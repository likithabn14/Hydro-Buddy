# 💧 Hydro Buddy

Hydro Buddy is a modern hydration reminder and water tracking web application built using **Python** and **Streamlit**.

It helps users develop healthy hydration habits by tracking daily water intake, monitoring progress, viewing weekly analytics, and earning achievements.

---
DEMO LINK:
https://hydro-buddy-8cebdbh4rekikepfran59c.streamlit.app/

## ✨ Features

- 💧 Track daily water intake
- ⏰ Reminder countdown timer
- 📊 Daily hydration progress
- 📈 Weekly analytics dashboard
- 🏆 Achievement badges
- 🔥 Hydration streak tracker
- 📋 Water intake history
- 📥 Download history as CSV
- 💡 Daily hydration tips
- 😊 Personalized Hydro Buddy emoji
- 🌙 Modern dark theme UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly Express
- Streamlit Auto Refresh
